Análise de dados com Python e Pandas

Neste projeto foi apresentada a biblioteca Pandas, uma biblioteca Python de código aberto para análise de dados. Ela dá ao Python a capacidade de trabalhar com dados do tipo planilha, permitindo carregar, manipular e combinar dados rapidamente, entre outras funções.

 **Assuntos abordados neste curso**:
 1. Python Básico
 1. Estrutura de dados
 1. Introdução a Biblioteca Pandas
 1. Trabalhando com arquivos em Excel
 1. Trabalhando com Datas
 1. Visualização de dados
 1. Análise Exploratória de dados
